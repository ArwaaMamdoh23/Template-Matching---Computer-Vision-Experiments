# Template Matching – Computer Vision Experiments

## Overview
This repository contains a set of computer vision experiments focused on Template Matching using OpenCV.  
The goal is to explore how template matching works in practice and how different approaches affect detection accuracy.

---

## Project Structure
This repository includes three assignments:

- **Assignment 1:** Basic template matching using OpenCV
- **Assignment 2:** Improved implementation with variations and parameter adjustments
- **Assignment 3:** Extended experiments and performance observations

---

## Concept
Template Matching is a technique used to locate a smaller image (template) within a larger image by sliding the template across the image and computing similarity scores.

It is commonly used in:
- Object detection (basic cases)
- Pattern recognition
- Image analysis tasks

---

## Techniques Used
- OpenCV template matching methods
- Normalized cross-correlation
- Image preprocessing
- Visualization of detection results

---

## Key Observations
- Performance depends heavily on lighting and scale
- Template matching is sensitive to rotation and scaling
- Different methods produce different detection accuracy
- Results improve with better preprocessing
